# Gizlilik Politikaları

Doğuş İpeksaç mobil uygulamalarının gizlilik politikalarının barındırıldığı statik site.

## Yapı

| Uygulama | Klasör | URL |
|----------|--------|-----|
| Ana sayfa | `/` | `index.html` |
| Daily Hydrate | `/dailyhydrate/` | `dailyhydrate/index.html` |
| Anı Yakala | `/capturethemoment/` | `capturethemoment/index.html` |
| App Plaka Kontrol | `/app_plaka_kontrol/` | `app_plaka_kontrol/index.html` |
| Kube Rush | `/kuberush/` | `kuberush/index.html` |

## Yeni uygulama ekleme

1. Yeni bir klasör oluşturun (ör. `my-app/index.html`)
2. Gizlilik politikasını ekleyin
3. Ana `index.html` dosyasındaki uygulama listesine yeni kartı ekleyin
